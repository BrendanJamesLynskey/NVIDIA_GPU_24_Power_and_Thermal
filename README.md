# Power & Thermal — Delivering 1000 W to a Card and Removing It Again

How a B200 burns 1000 watts and gets that power and heat managed: 12V-2x6 cabling, multi-phase VRMs, multi-rail design, transient response, GPU Boost and P-states, power capping, vapour chambers, direct-to-chip liquid cooling, immersion. Includes datacenter row-level engineering and an interactive power-and-cooling calculator.
